Implementation Notes

SSE & Real-time Updates

The issue_updated SSE event should trigger a refresh of both the progress bar and the stages checklist. Use HTMX hx-get with hx-trigger="issue-changed from:body" on the stage containers to enable surgical updates without page reloads.

Manual Toggling

Each checkbox in the sidebar should trigger a PATCH /api/v1/issues/{key} request. The payload should include the full updated stages array. A successful patch should then broadcast the issue_updated event via SSE.
